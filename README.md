

## Technical Challenge for LookingBus

## Instructions
1)	Using 511.org choose a bus, show the google map with the bus on the map and the bus icon where its location is.
2) Collect the location of the bus along the route by querying the 511.org every minute (can make the bus icon movement animated)
3) Pull location of stops, show stops on the map.
4)	Show the average speed of the bus between stops 
-	You can choose to show the velocity distribution between the stops

## Technologies
- HTML
- CSS
- Javascript
- JQuery
- [Chart.js](https://www.chartjs.org/)
- Google Maps JavaScript API
- [511 SF Bay Area Public Transit Open Data](https://511.org/open-data)
